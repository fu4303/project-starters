# React Movie Finder

## Description:

Movie Finder Web App built with React using TMDB API

## Tools & Languages:

- React
- React Router
- React Tabs
- React Context API
- React Hooks
- TheMovieDB API
- Axios
- Bootstrap Material Design
- Node Sass
